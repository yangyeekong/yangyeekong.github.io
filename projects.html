Hexapod

I bought the main body, boards, motors, and sensors online from Freenove. Their documentation covers building and Raspberry Pi setup. For this project I used a Raspberry Pi 5 with 16 GB of RAM and a 64 GB micro SD card.

What it takes to build

Hardware

  • Arduino Mega or Uno
  • Raspberry Pi
  • Ultrasonic sensor (for obstacle detection)
  • IMU (gyroscope + accelerometer for balance)
  • Optional: Camera module (for vision)
  • Rechargeable batteries
  • 12 servo motors
  • PCA9685 (16-channel PWM driver for servos)
  • Jumper wires, servo extension cables
  • Breadboard or custom PCB
  • Screws, nuts, bolts, servo mounting brackets

Frame materials

  • Acrylic sheets (lightweight)
  • Aluminum (durable)
  • 3D-printed parts (PLA/ABS)
  • Wood or cardboard (prototypes)

Code, tools and software

  • Python (Raspberry Pi control, sensors, AI)
  • Arduino (C/C++) for low-level servo control
  • Command line / Bash for Pi setup
  • Fusion 360 / SolidWorks for mechanical design
  • Screwdrivers, pliers, soldering iron, 3D printer (optional)
