Hexapod


 What it takes to build:

Hardware:
- Arduino Mega or Uno
- Raspberry Pi
- Ultrasonic sensor (for obstacle detection)
- IMU (gyroscope + accelerometer for balance)
- Optional: Camera module (if using Raspberry Pi for vision)
- Rechargeable batterys
- 12 servo motors
- PCA9685 (16-channel PWM driver for servos)
- Jumper wires, servo extension cables
- Breadboard or custom PCB for connections
- Screws, nuts, bolts
- Servo mounting brackets

Frame Materials:
- Acrylic sheets (lightweight and strong)
- Aluminum (for durability)
- 3D-printed parts (PLA or ABS)
- Wood or cardboard (for prototypes)

Code Languages, Tools, and Software
- Python (for Raspberry Pi control, AI, and sensor integration)
- Arduino IDE (C/C++) (for servo control and basic logic)
- Command Line / Bash (for Raspberry Pi setup)
- Software for Robot Locomotion

Mechanical Design Software:
- Fusion 360 or SolidWorks (for designing body parts)

Tools:
- Screwdrivers, pliers, soldering iron, 3D printer (optional but useful)

Testing & Debugging Gear:
- Multimeter
- USB cables for programming

I bought the main body, boards, moters, and sensors online from Freenove. This is also where they have the documentation for building and setting up software.

For this project I used a Raspbery Pi 5 with 16gb of RAM. I used a 64gb micro SD card for storage
